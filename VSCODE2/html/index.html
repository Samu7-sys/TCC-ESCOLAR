Oque é HTML?

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto),
é a linguagem fundamental para estruturar o conteúdo de páginas web. Ela usa tags para
definir elementos como parágrafos, títulos, listas, imagens e outros componentes,
determinando como esses elementos serão exibidos em um navegador.

Oque é CSS

CSS, que significa Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem
usada para estilizar elementos HTML em páginas web. Ele define a aparência visual, como cores,
fontes, tamanhos, espaçamento, e layout das páginas. Em resumo, o CSS separa a estrutura (HTML)
do conteúdo (CSS) da página, permitindo personalizar o visual de forma consistente e eficiente.

Oque é JS (Java Script)

JavaScript é uma linguagem de programação de script de alto nível, amplamente utilizada para criar
interatividade em páginas web e aplicativos web. Ela permite adicionar comportamento dinâmico, como
animações, interação com o usuário, validação de formulários e muito mais.
